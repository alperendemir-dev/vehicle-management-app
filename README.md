# Vehicle Management App (C# WPF)

Diese Anwendung wurde im Rahmen meiner Umschulung zum Fachinformatiker für Anwendungsentwicklung entwickelt.

## Funktionen
- Hinzufügen von Fahrzeugen
- Validierung von Eingaben (z. B. Kilometerstand und Preis)
- Anzeige und Verwaltung von Fahrzeugdaten
- Übersichtliche Darstellung in einer Benutzeroberfläche

## Technologien
- C#
- WPF (Windows Presentation Foundation)
- .NET

## Ziel des Projekts
Ziel war es, eine praxisnahe Desktop-Anwendung zu entwickeln und grundlegende Konzepte wie Datenstrukturen, Eingabevalidierung und Benutzeroberflächen umzusetzen.
